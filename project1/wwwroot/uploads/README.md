# 📁 תיקיית Uploads

## מטרה
תיקייה זו נוצרה עבור העלאת תמונות למנהל.

## מבנה
```
uploads/
├── products/      ← תמונות מוצרים (קדמית + אחורית)
├── categories/    ← תמונות קטגוריות
└── styles/        ← תמונות סגנונות
```

## ⚠️ הערה חשובה
**תיקייה זו נוספה במיוחד עבור העלאת תמונות למנהל.**

אם את רוצה למחוק את כל התכונה הזו בעתיד:
1. מחקי את כל התיקייה `uploads/`
2. חפשי "נוסף עבור העלאת תמונות למנהל" בקבצי ה-Controllers
3. מחקי את הקוד המסומן
